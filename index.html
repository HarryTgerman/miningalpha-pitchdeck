Mining data in 2026 is where on-chain data was in 2019.

Before Dune Analytics, on-chain data was technically public but practically unusable — raw logs scattered across nodes, accessible only to engineers.

Dune turned it into SQL tables. Suddenly anyone could query wallets, flows, and protocols in seconds. Dashboards, funds, and bots were built on top.

We’re doing the same for mining. MiningAlpha transforms NI 43-101 and JORC filings into structured, machine-queryable supply data — the base layer for the entire industry.
